# Weather App

Если кратко пробежаться, приложение даёт юзерам возможность смотреть погодные данные с точки, которая указана в полях координатов, а именно: 1) температура, 2) статус погоды(тип осадки солнечность облачность), 3) Ветер, 4) Местное время, 5) Часовой пояс. Юзер может смотреть указанную точку на интерактивной карте. Так же есть возможность добавлять несколько виджетов с погодой, которые пишуться в localStorage для удобства. Приправлено накинутыми стилями чтобы не смотреть на вырвиглазный html дефолт (макет вообще не стилизован). Присутствует валидация данных, так что в теории ничего не должно упасть, всё обрабатывается спокойно.
